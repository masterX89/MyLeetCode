# [103.带环链表 II](https://www.lintcode.com/problem/103/)

## 题目

给定一个链表，如果链表中存在环，则返回到链表中环的起始节点，如果没有环，返回null。

样例
样例 1：

输入：
```
null，no cycle
```
输出：
```
no cycle
```
解释：

链表为空，所以没有环存在。

样例 2：

输入：
```
21->10->4->5，tail connects to node index 1
```
输出：
```
10
```
解释：

最后一个节点5指向下标为1的节点，也就是10，所以环的入口为10。
